## Installation ‚è≥
### Using npm
```sh
npm install kvass
```
### Using yarn
```sh
yarn add kvass
```

### Using direct CDN link
<p>Just add this link to the <head> tag in your HTML file</p>
  
`<link rel="stylesheet" href="https://unpkg.com/kvass@1.0.1/kvass.css">`
